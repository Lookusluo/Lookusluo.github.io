AWS remote access archive

Posted on 2016-05-09

##1.Access

First open my terminal and cd to cwd of key’s file, then input

ssh -i “*.pem” ec2-user@public_IP

Now you access AWS server

##2.Archive

Go to public file cd /var/www/html/

and archive any folder or file by inputing zip -r file_name.zip file_name

after zip you can down file by input :
scp -i “*.pem” ec2-user@Public_IP:/var/www/html/file_name.zip pwd_of_download_destination

Then End
